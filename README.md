# codecept
# Introduction
This project demonstrates testing REST API requests using CodeceptJS, Chai, Javascript.

# Installation
cd codecept<br>
npm i

# How to trigger API tests & generate report
npx codeceptjs run --reporter mochawesome

# Reports path
\codecept\mochawesome-report
